HOW TO PLAY

1. Select your difficulty level and start the game.
2. Once the number of levels is displayed, guess the minimum number of coins needed to create a square in just 3 tries.
3. 3 wrong guesses will display the correct answer.
4. if number of coins are guessed correctly, then you win and game ends.
